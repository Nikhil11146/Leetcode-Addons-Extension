# 🚀 LeetCode Addons

A lightweight Tampermonkey userscript that enhances your LeetCode profile with useful statistics and quality-of-life improvements.

## ✨ Features

### 📈 Questions Per Day

Automatically calculates and displays:

```text
Questions Per Day = Total Solved Questions / Total Active Days
```

This helps you track your long-term consistency rather than focusing only on total solved problems.

Example:

```text
Questions/day: 1.66
```

Displayed directly on your LeetCode profile alongside your existing stats.

## 🛠 Installation

### Step 1: Install Tampermonkey

Download and install Tampermonkey for your browser:

* Chrome
* Edge
* Firefox
* Brave

Official Website:

https://www.tampermonkey.net/

---

### Step 2: Install the Script

Install from GreasyFork:

**GreasyFork Link:**
`<ADD_GREASYFORK_LINK_HERE>`

Click **Install** and Tampermonkey will automatically add the script.

---

### Step 3: Open Your LeetCode Profile

Visit:

```text
https://leetcode.com/u/<your-username>/
```

The script will automatically inject the additional statistics.

## 🎯 Why Use This?

LeetCode shows:

* Total solved questions
* Active days
* Max streak

But it doesn't show how consistently you're solving questions over time.

This addon fills that gap by providing a quick productivity metric at a glance.
